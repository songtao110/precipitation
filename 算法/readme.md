## 二叉堆
```
二叉堆就是一种完全二叉树，所以适合存储在数组中，而且二叉堆拥有一些特殊性质。
二叉堆的操作很简单，主要就是上浮和下沉，来维护堆的性质（堆有序），核心代码也就十行。
优先级队列是基于二叉堆实现的，主要操作是插入和删除。
**插入**是先插到最后，然后上浮到正确位置；
**删除**是调换位置后再删除，然后下沉到正确位置
```
